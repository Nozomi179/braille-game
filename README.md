# 点字学習ゲーム (Braille Learning Game)

このゲームは、点字のパソコン入力を楽しく学べるWebアプリケーションです。

## 遊び方
1. 画面上部のボタンで行を選択して「START」ボタンを押してください。
2. 画面上から落ちてくるボールに書かれた文字に対応する点字を、キーボードの「f, d, s / j, k, l」キーを使って入力します。
3. 6点式点字の入力方法を、ゲームを楽しみながら自然に習得できます。

## 操作方法
キーボードの以下のキーを点字の6点に対応させています。

| 左列 (1, 2, 3) | 右列 (4, 5, 6) |
| :--- | :--- |
| **f** (1) | **j** (4) |
| **d** (2) | **k** (5) |
| **s** (3) | **l** (6) |

※ 入力後に「Enter」または「Space」キーで回答を確定してください。

## 点字のルールについて
このゲームでは、学習しやすくするために濁音や拗音をステップに分けて入力する方式を採用しています。

■ 濁音「が」の例
このゲームでは学習ステップとして以下のように入力します。

1. 濁音符 (5の点) を入力 ── [ k ] キー
2. 文字「か」 (1, 6の点) を入力 ── [ f ] + [ l ] キー
