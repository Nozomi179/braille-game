# 点字学習ゲーム(Braille learning Game)

このゲームは、点字のパソコン入力を楽しく学べるWebアプリケーションです。

## 遊び方
- 画面上部のボタンで行を選択して「START」ボタンを押してください。
- 画面上から落ちてくるボールに書かれた文字に対応する点字を、キーボードの「f, d, s / j, k, l」キーを使って入力します。
- 6点式点字の入力方法を、ゲームを楽しみながら自然に習得できます。

## 操作方法
キーボードの以下のキーを点字の6点に対応させています。

| 左列 (1, 2, 3) | 右列 (4, 5, 6) |
| :--- | :--- |
| **f** (1) | **j** (4) |
| **d** (2) | **k** (5) |
| **s** (3) | **l** (6) |

*入力後に「Enter」または「Space」キーで回答を確定してください。*
